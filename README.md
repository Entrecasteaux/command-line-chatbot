# command-line-chatbot

ChatGPT designed for use in the command line

## Usage

```bash
$ go install Entrecasteaux.de/x/command-line-chatbot@latest
$ echo $OPENAI_API_KEY # this should be the OpenAI API key
$ command-line-chatbot
```
